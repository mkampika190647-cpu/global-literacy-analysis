# Global Literacy Rate Analysis (1970–2024)
> **A Multiple Linear Regression Study on Educational Resources and Literacy Outcomes**

โปรเจกต์วิเคราะห์ปัจจัยเชิงโครงสร้างและทรัพยากรทางการศึกษาที่มีอิทธิพลต่ออัตราการรู้หนังสือของประชากรโลก ครอบคลุมข้อมูลย้อนหลังกว่า 50 ปี เพื่อระบุตัวแปรสำคัญที่ช่วยยกระดับทักษะพื้นฐานของประชากรในระดับนานาชาติ

## 📌 บทสรุปโครงการ (Project Overview)
* **วัตถุประสงค์ (Objective):** เพื่อศึกษาความสัมพันธ์และวิเคราะห์ปัจจัยด้านทรัพยากรทางการศึกษาที่มีอิทธิพลต่ออัตราการรู้หนังสือ พร้อมสร้างแบบจำลองทางสถิติเพื่ออธิบายแนวโน้มที่เกิดขึ้น
* **ปัญหา (Problem):** ความแตกต่างของอัตราการรู้หนังสือระหว่างประเทศกำลังพัฒนาและประเทศพัฒนาแล้ว ยังคงเป็นอุปสรรคสำคัญต่อการพัฒนาทรัพยากรมนุษย์และเศรษฐกิจโลก
* **ข้อมูล (Data):** ข้อมูลทุติยภูมิจากธนาคารโลก (World Bank - WDI) และองค์การยูเนสโก (UNESCO - UIS) ครอบคลุมช่วงปี ค.ศ. 1970 - 2024

## 🛠️ เทคนิคและเครื่องมือ (Tech Stack)
* **Tools:** Python, AI Studio (RapidMiner)
* **Statistical Methods:**
    * **Descriptive Statistics:** วิเคราะห์ลักษณะทั่วไปและการกระจายของข้อมูล
    * **Bivariate Analysis:** วิเคราะห์สหสัมพันธ์ (Correlation) รายคู่ระหว่างตัวแปร
    * **Multiple Linear Regression:** แบบจำลองหลักเพื่อวิเคราะห์อิทธิพลของหลายตัวแปรพร้อมกัน
* **Metrics:** ค่า R-squared (R2), Adjusted R2, และ p-value สำหรับประเมินความเหมาะสมของแบบจำลอง

## 📊 ขั้นตอนการดำเนินงาน (Methodology)
1. **Data Preprocessing:** รวบรวมข้อมูลจากแหล่งนานาชาติ จัดการค่าที่ขาดหาย (Missing Data) และตรวจสอบค่าผิดปกติ (Outliers)
2. **Univariate Analysis:** พบว่าตัวแปรจำนวนเด็กนอกระบบการศึกษามีการกระจายแบบเบ้ขวาอย่างรุนแรง และมีค่าความแปรปรวนสูงมาก
3. **Bivariate Analysis:** พบสหสัมพันธ์เชิงบวกระดับสูงระหว่างอัตราการรู้หนังสือกับงบประมาณด้านการศึกษา (r=0.715) และอัตราการสำเร็จการศึกษาประถม (r=0.699)
4. **Multivariate Modeling:** สร้างสมการถดถอยพหุคูณเพื่อประเมินน้ำหนักของแต่ละปัจจัยพร้อมกัน

## 🚀 ผลการศึกษาและข้อเสนอแนะ (Results & Insights)
* **Key Discovery:** อัตราการสำเร็จการศึกษาระดับประถมศึกษา และ งบประมาณด้านการศึกษาของรัฐบาล คือปัจจัยที่มีอิทธิพลเชิงบวกต่อการเพิ่มอัตราการรู้หนังสืออย่างมีนัยสำคัญที่สุด
* **Model Insight:** แบบจำลองสามารถอธิบายความแปรปรวนของอัตราการรู้หนังสือได้ประมาณ 54.1% (R2 = 0.541)
* **Strategic Suggestion:** นโยบายควรเน้นไปที่การเพิ่มงบประมาณเพื่อพัฒนาคุณภาพการศึกษาขั้นพื้นฐานมากกว่าการเน้นแค่การเข้าถึงระบบเพียงอย่างเดียว
